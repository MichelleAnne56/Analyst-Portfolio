# Introduction to Chinese Value Added Trade Project  
My second research product is an analysis of the value added trade in China's exports using the OCED's Trade in Value (TiVA) database. In 2022, 83 percent of the value in Chinese exports originated in China, which suggests that China's domestic industry remains quite strong. Also, no single country accounted for more than five percent of the value added in Chinese exports, underscoring China's effort to avoid over-reliance on any individual partner. The top nine sources of value added were the United States, Chinese Taipei, South Korea, Japan, Australia, Russia, Saudi Arabia, Germany, and the category for the rest of the world. 

## Approach 
I calculated the value added in China's exports as a percentage of total value added to see which countries have contributed the most to Chinese supply chains. I used the percentage of total value added rather than nominal figures to account for inflation and to identify broader geopolitical trends.
