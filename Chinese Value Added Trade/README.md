# Introduction to Chinese Value Added Trade Project  
My second research product is an analysis of the value added trade in China's exports using the OCED's Trade in Value (TiVA) database. The TiVA dataset maps the value added by each country in the production of goods and services consumed worldwide. The analytical advantage of value added trade data is that it captures both the direct and indirect value added in trade, allowing analysts to map how supply chains are routed around the globe.

## Approach 
I calculated the value added in China's exports as a percentage of total value added to see which countries have contributed the most to Chinese supply chains. I used the percentage of total value added rather than nominal figures to account for inflation and to identify broader geopolitical trends.
